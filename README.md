# CodeAlpha_Iris_Flower_Classification

This repository contains my submission for the CodeAlpha Machine Learning Internship task.

## Project Overview
The goal of this project is to classify Iris flowers into three species (*Setosa*, *Versicolor*, and *Virginica*) based on measurements of their sepals and petals.

## Tech Stack
* Python
* Pandas(Data Manipulation)
* Scikit-Learn (Decision Tree)

## Results
The trained model achieved an accuracy score of 1.0 on the test dataset.
